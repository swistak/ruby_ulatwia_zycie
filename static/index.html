Linux

Linux daje trzy możliwości instalacji Ruby:

  • Z pakietów
  • Przy pomocy RVM
  • Ze źródeł

Linux - Pakiety

  • apt-get install ruby # Debian/Ubuntu
  • yum install ruby # RedHat/CentOS

Linux - RVM

$ curl -L get.rvm.io | bash -s stable
$ source ~/.bash_profile
$ rvm install 1.9.3
$ rvm 1.9.3
